# Game sounds

Place your audio files here. The game will play:

- **shoot.mp3** — when the player shoots (click or Space)
- **reload.mp3** — when the next set of targets appears

Use **.mp3** or **.ogg**; the game looks for `.mp3` first.  
If a file is missing, that sound is simply skipped (no error).
